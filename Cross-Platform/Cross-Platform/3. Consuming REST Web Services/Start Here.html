Before you begin

This class uses labs based on Xamarin.Forms to demonstrate cross-platform capabilities on iOS, Android and Windows Phone. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This is the first of two classes on consuming web services in your mobile applications. This class covers the available network types and how to detect which ones are available to your application and then dives into consuming REST services with HttpClient.

Lecture

XAM150 - Cross Platform RESTful Web Services.

Objectives

  1. Obtain the device's network capabilities
  2. Introduction to REST
  3. Consuming REST services with Xamarin
  4. Integrating with the platform

Exercises

  1. Determine the network connectivity
    Exercise
    In this exercise, you will create a Xamarin.Forms application to detect the current network state for iOS, Android or Windows Phone and display relevant detals about the connection.
  2. Communicating with a Book Service
    Exercise
    Next, you will create a Xamarin.Forms application to consume an existing REST service which manages a library of books. You will authenticate with the service, retrieve existing books, add new books, update books and delete books from your bookshelf.
  3. Adding support for ModernHttpClient
    Exercise
    In this final group exercise, we will modify the bookshelf client to use the ModernHttpClient component to improve the performance of the network activity.
