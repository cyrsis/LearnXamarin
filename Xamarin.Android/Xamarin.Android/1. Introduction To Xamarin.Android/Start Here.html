Before you begin

This is an Android class. To complete the coding exercises, you will need an Android development environment with either an emulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

This course covers all the core concepts you will need to get a simple Xamarin.Android app written and tested. You will see how to create a new project, build a UI in XML, and write C# code logic to drive the UI. You will also learn how to use the Android tools to keep your development environment up-to-date with the latest versions of Android.

Objectives

  1. Create a Xamarin.Android project
  2. Decompose an app into Activities
  3. Build an Activity's UI
  4. Write an Activity's behavior
  5. Update your Android SDK

Lecture

AND101 - Introduction to Android.

Exercises

  1. Create a Xamarin.Android project
    Exercise
    You will begin by creating a new Xamarin.Android project in your choice of IDE.
  2. Add views to a layout file manually and with the Designer tool
    Exercise
    Here you will build a simple UI. You'll do part of it by hand-coding XML and part using the Designer tool.
  3. Implement an Activity's behavior and run your app in an emulator
    Exercise
    In this exercise, you will write C# code to access UI views and perform some work on user-entered data.
  4. Update Tools and SDK Platform
    Exercise
    Finally, you will use the Android SDK Manager to update the Android SDK and tooling on your development machine.
