Before you begin

This is an Android class. To complete the coding exercises, you will need an Android development environment with either an emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

Introduces several common user-interface navigation metaphors for Android. Also discusses the most appropriate uses for each and how to implement them in Xamarin apps.

Objectives

  1. Implement Stack navigation
  2. Introduce Fragments
  3. Implement Tab navigation
  4. Introduce ActionBar
  5. Implement Drawer navigation

Lecture

AND205 - Android Navigation

Exercises

  1. Stack Navigation
    Exercise
    In this exercise, we will see how Android handles navigation stacks.
  2. Tab Navigation
    Exercise
    The goal of this lab will be to implement the tab navigation pattern.
  3. Drawer Navigation
    Exercise
    The lab will show how to use Drawer navigation in Android.
