Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This class is designed to introduce you to the features and architecture of Xamarin.Forms, a cross-platform UI toolkit for iOS, Android and Windows Phone. We will be covering basic app creation, the layout model, using platform-specific features and basic navigation.

Lecture

XAM120 - Introduction to Xamarin.Forms.

Objectives

  1. Cross-Platform UI Strategies
  2. Xamarin.Forms App Structure
  3. Pages, Controls, and Layout
  4. Using Platform-Specific Features

Exercises

  1. Creating our first Xamarin.Forms application
    Exercise
    In this exercise, we will create a basic Xamarin.Forms application together using the built-in project templates and go through the solution structure.
  2. Creating Xamarin.Forms Phoneword
    Exercise
    In this exercise, you will create a new Xamarin.Forms application (Phoneword) to enter an alphanumeric phone number which is then translated to a standard numeric telephone number.
  3. Adding support for dialing the phone
    Exercise
    In this final exercise, you will extend your Phoneword application by adding support to dial the phone and display an alert view cross-platform.
