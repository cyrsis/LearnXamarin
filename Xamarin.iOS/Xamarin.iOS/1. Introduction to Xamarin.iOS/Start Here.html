Before you begin

This is an iOS class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either a configured iOS Simulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This class will introduce you to iOS development with the Xamarin toolset. It covers application fundamentals including: the Model-View-Controller pattern, creating screens and interacting with controls and providing application behavior.

Lecture

iOS101 - Introduction to Xamarin.iOS.

Objectives

  1. Introduce the development tools
  2. (De)constructing the application
  3. Adding views and behavior

Exercises

  1. Creating and running your first iOS application
    Exercise
    In this exercise, you will create a simple iOS application using the Single-View application template and then run it using the iOS Simulator.
  2. Add a root view controller to the Tip Calculator app
    Exercise
    Next, you will add a new UIViewController to the application and assign it as the RootViewController for the app.
  3. Create the UI for a Tip Calculator
    Exercise
    In this exercise, you will create the user-interface for a simple Tip Calculator.
  4. Add logic to your Tip Calculator
    Exercise
    In this final exercise, you will add the behavior to make your tip calculator application fully-functional.
  5. Homework
    Exercise
    In this optional post-class exercise, you will add some additional logic to your Tip Calculator to make the tip amount configurable.
