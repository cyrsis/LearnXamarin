Before you begin

This is an iOS class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either a configured iOS Simulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This course provides an introduction to using the iOS Storyboard Designer. During this session you will gain knowledge and experience using many of the features the iOS Designer offers including: UIViews, UIViewControllers, auto layouts, and creating segues and navigation.

Lecture

iOS102 - Getting Started with the iOS Designer

Objectives

  1. Create a single screen application and add controls
  2. Describe and use Auto Layouts
  3. Add behavior to controls programmatically
  4. Apply segues and navigation

Exercises

  1. Create and layout a single view application using the Xamarin.iOS Designer
    Exercise
    In this exercise, you will create a simple iOS application using the Single-View application and add views and controls to it using the iOS Designer.
  2. Use Auto Layouts in your application
    Exercise
    In this exercise, you will add constraints to your views using the iOS Designer.
  3. Code behaviors for your app
    Exercise
    In this exercise, you will code behaviors to the controls of your app to add interactivity and functionality.
  4. Add a second screen to your app and code a button to navigate to it
    Exercise
    In this exercise, you will add a second screen to the storyboard and create buttons to present and dismiss a modal view controller in code.
  5. Add segues to define the navigation
    Exercise
    In this final exercise, you will add segues to define the navigation of the app directly from the Designer.
