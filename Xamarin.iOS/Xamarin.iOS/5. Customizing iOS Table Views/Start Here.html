Before you begin

This course is specific to iOS development. In order to complete the coding exercises you will need an iOS development environment with either a simulator or a device to run the exercise. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This course focuses on customizing table view cells in iOS.

Objectives

  1. Customize table view cells in code
  2. Customize table view cells in the designer
  3. Group content in the table view

Lecture

IOS115 - Customizing Table Views

Exercises

  1. Customize a default table cell
    Exercise
    In this exercise, we will be customizing the appearance of a the default table view cells.
  2. Create a custom table view cell in code
    Exercise
    In this exercise, we will create a custom table view cell programmatically.
  3. Create static cells in the designer
    Exercise
    In this exercise, we will design static cells for a settings page for an app utilizing the iOS storyboard designer.
  4. Create a prototype table view cell using the designer
    Exercise
    In this exercise, we will create a custom dynamic prototype cell and wire it up to display weather data.
  5. Create a grouped table with an index
    Exercise
    In this exercise, we will enable grouping in an iOS Table View and use the index with grouping for quicker navigation.
